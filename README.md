# Web Tools - 在线工具集合网站

简洁、高效的在线工具集合网站，专注于提供开发者日常工作中最常用的核心功能。

## 功能特性

- ✅ JSON格式化、解析
- ✅ SQL格式化
- ✅ 时间戳转换
- ✅ Unicode编码解码

## 技术栈

- **React 18+** - UI框架
- **TypeScript 5+** - 类型系统
- **Vite 5+** - 构建工具
- **Tailwind CSS** - 样式框架
- **Monaco Editor** - 代码编辑器
- **React Router 6+** - 路由管理
- **dayjs** - 时间处理
- **sql-formatter** - SQL格式化

## 快速开始

### 安装依赖

```bash
npm install
```

### 开发模式

```bash
npm run dev
```

### 构建生产版本

```bash
npm run build
```

### 预览生产版本

```bash
npm run preview
```

## 项目结构

```
web-tools/
├── doc/              # 文档目录
├── public/           # 静态资源
├── src/
│   ├── components/   # 公共组件
│   ├── pages/        # 页面组件
│   ├── hooks/        # 自定义Hooks
│   ├── utils/        # 工具函数
│   ├── styles/       # 样式文件
│   └── types/        # 类型定义
└── package.json
```
