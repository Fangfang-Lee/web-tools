# AI助手项目指南 (AGENTS.md)

> 本文档用于帮助AI助手理解项目，协助开发工作。

---

## 项目概述

**项目名称**: Web Tools - 在线工具集合网站  
**版本**: v1.0  
**项目类型**: 纯前端Web应用  

### 项目简介

简洁、高效的在线工具集合网站，专注于提供开发者日常工作中最常用的核心功能。纯前端架构，所有功能在浏览器端完成。

### 项目定位

- **目标用户**: 开发者、技术人员
- **核心价值**: 提供常用开发工具，提升工作效率
- **设计理念**: 简洁、高效、专注核心功能

---

## 项目目标

1. **功能目标**: 提供4个核心工具（JSON格式化、SQL格式化、时间戳转换、Unicode编码解码）
2. **性能目标**: 页面加载 < 2秒，转换响应 < 100ms
3. **用户体验目标**: 界面简洁、操作直观、错误提示友好
4. **技术目标**: 代码质量高、类型安全、易于维护

---

## 核心功能

### 1. JSON格式化、解析
- JSON格式化（美化、缩进，可配置2/4空格）
- JSON压缩
- JSON解析（实时解析、错误提示）
- JSON验证

### 2. SQL格式化
- SQL格式化（自动格式化、缩进对齐）
- SQL压缩
- 关键字大小写转换（UPPER/lower）
- 支持多SQL方言（MySQL、PostgreSQL、SQL Server、Oracle、SQLite）

### 3. 时间戳转换
- 时间戳转日期（支持秒级/毫秒级）
- 日期转时间戳
- 时区支持（UTC、GMT+8等）
- 多种日期格式（ISO 8601、本地格式）

### 4. Unicode编码解码
- Unicode编码（\uXXXX、U+XXXX、十进制格式）
- Unicode解码（自动识别格式）
- 字符信息显示（码点、名称、分类）

## 开发规范

### 命名规范
- **组件**: PascalCase (`JsonFormatter.tsx`)
- **函数/变量**: camelCase (`formatJson()`)
- **常量**: UPPER_SNAKE_CASE (`MAX_TEXT_LENGTH`)
- **类型/接口**: PascalCase (`JsonFormatterOptions`)

### TypeScript规范
- 严格模式
- 所有函数必须有类型注解
- 公共API必须有JSDoc注释
- 避免使用 `any`

### React规范
- 使用函数式组件 + Hooks
- Props接口命名：`ComponentNameProps`
- 优先使用 `const`

### Git规范
- 分支: `feature/xxx`, `fix/xxx`
- 提交: 遵循Conventional Commits (`feat:`, `fix:`, `docs:`等)

## 设计要求

### UI设计原则
1. **简洁性**: 界面简洁，避免冗余元素
2. **一致性**: 统一的视觉风格和交互模式
3. **易用性**: 操作直观，无需学习成本
4. **响应式**: 支持桌面端和移动端

### 布局设计
- 工具页面：左右分栏（输入/输出）
- Header: 导航栏
- Main: 工具内容区域
- Footer: 版权信息

### 颜色方案
- 主色: #1890ff
- 成功: #52c41a
- 警告: #faad14
- 错误: #f5222d

### 响应式断点
- 移动端: < 768px
- 平板: 768px - 1023px
- 桌面端: >= 1024px

## 注意事项

### 开发注意事项

1. **性能优化**
   - 大文本处理使用防抖/节流
   - 路由使用懒加载
   - 避免不必要的重渲染

2. **错误处理**
   - 所有用户输入必须验证
   - 错误提示要友好、清晰
   - 使用try-catch捕获异常

3. **用户体验**
   - 操作要有即时反馈
   - 复制成功要有提示
   - 加载状态要显示

4. **安全性**
   - 所有处理在客户端完成
   - 不收集用户数据
   - 输入验证防止XSS

5. **兼容性**
   - 支持现代浏览器
   - 使用标准API
   - 测试不同浏览器

### 代码质量要求

1. **类型安全**: 避免使用 `any`，类型定义完整
2. **代码复用**: 提取公共组件和工具函数
3. **可维护性**: 代码结构清晰，函数职责单一
4. **测试**: 核心工具函数要有单元测试，覆盖率 > 80%
